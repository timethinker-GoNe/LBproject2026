# 개발일지 2026-06-16

## 1. 대화창 캐릭터 캐리커쳐 UI 구현

### 배경
기존 `DialoguePanel`은 대화창 내부 좌측에 소형 portrait를 표시하는 구조였음.
비주얼노벨 스타일(화면 좌/우 풀사이즈 캐릭터 이미지)로 교체 요청.

### 변경 파일
- `Assets/DialogueQuests/Scripts/Data/ActorData.cs`
- `Assets/DialogueQuests/Scripts/UI/DialoguePanel.cs`

### ActorData.cs 변경
`PortraitSide` 열거형과 풀 portrait 관련 필드 추가:
```csharp
public enum PortraitSide { Left, Right }

// [Header("Full Portrait")] 아래
public Sprite portrait_full;       // 화면 옆에 표시할 풀사이즈 이미지
public PortraitSide portrait_side; // 좌/우 위치 (캐릭터별 고정)
```

### DialoguePanel.cs 변경
추가된 Inspector 필드:
```csharp
// [Header("Full Portrait")]
public Image portrait_full_left;
public Image portrait_full_right;
public Image portrait_name_bg_left;   // 이름 배경 박스
public Image portrait_name_bg_right;
public Text  portrait_name_left;      // 이름 텍스트
public Text  portrait_name_right;
```

동작:
- 대화 시작 시 `msg.actor.portrait_side`에 따라 left/right 중 하나 활성화
- 이름은 대화창 내부 title_box 대신 portrait 위에 표시
- 대화 종료(`Hide()` override) 시 모두 비활성화

### Unity 에디터 설정 (수동)
`DQUICanvas` 아래 계층 구조:
```
DQUICanvas
├── PortraitFullLeft          ← Image 컴포넌트 (Color Alpha=0로 에디터에서 숨김)
│   ├── PortraitNameBgLeft    ← Image (이름 배경)
│   └── PortraitNameLeft      ← Text (이름)
├── PortraitFullRight         ← 동일 구조
│   ├── PortraitNameBgRight
│   └── PortraitNameRight
├── DialoguePanel             ← 대화창 (portrait보다 아래 → 앞에 렌더링)
└── ...
```
`DialoguePanel` 컴포넌트 Inspector에서 6개 필드 연결 필요.

각 NPC `ActorData` ScriptableObject에서:
- `Portrait Full` → 풀사이즈 sprite 할당
- `Portrait Side` → Left / Right 선택

## 2. PNG → Sprite 변환 (asha 캐리커쳐)

## 3. 아이콘 텍스처 압축 설정 (sugar_tester)

`Assets/FarmingEngine_study/icon/sugar_tester.png.meta`

Standalone 플랫폼 압축 품질 개선:

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| `textureCompression` (Standalone) | 1 (Low) | 2 (Normal) |

- 설정 위치: Unity Inspector → PNG 선택 → 하단 **PC, Mac & Linux Standalone** 탭 → Compression
- 퀵슬롯바 아이콘이 흐릿하게 보이는 원인이었음
