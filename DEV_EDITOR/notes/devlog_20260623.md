## 1. FullInventoryPanel 클릭/이동 버그 수정

### 증상
디자인편집기(project_structure.html) fip-zone에서 중앙 클릭으로 활성화 및 이동이 안 됨.  
가장자리(리사이즈 핸들) 드래그는 정상 동작. 퀵슬롯바/장비창은 정상.

### 원인 분석
`initZones()`에서 `updateZoneFromData(p.key)` 에러 발생 시 `z.addEventListener` 라인에 도달하지 못해 fip-zone의 mousedown 리스너가 아예 미등록될 가능성 + 전역 mousedown 핸들러가 리사이즈 핸들만 처리하고 이동은 처리 안 함.

### 변경 파일
`DEV_EDITOR/project_structure.html`

### 수정 내용

**Fix 1 — initZones try-catch** (line ~5275):
```javascript
// 변경 전
updateZoneFromData(p.key);

// 변경 후
try{updateZoneFromData(p.key);}catch(err){console.warn('[initZones]',p.key,err);}
```
에러가 나도 `z.addEventListener` 라인은 반드시 실행됨.

**Fix 2 — 전역 document mousedown 핸들러 확장** (line ~5566):
```javascript
// 변경 전: resize handle(data-h)만 처리
document.addEventListener('mousedown',function(e){
  var h=e.target.dataset&&e.target.dataset.h; if(!h)return;
  ...resize path only...
});

// 변경 후: resize + move 모두 처리
document.addEventListener('mousedown',function(e){
  var h=e.target.dataset&&e.target.dataset.h;
  var zEl=e.target.closest('[data-panel]'); if(!zEl)return;
  var key=zEl.getAttribute('data-panel');
  var sr=document.getElementById('uid-screen').getBoundingClientRect();
  var d=panelData[key]; if(!d)return;
  e.preventDefault();
  selectPanel(key);
  if(h){
    e.stopPropagation();
    drag={type:'resize', ...};
    e.target.classList.add('active');
  } else {
    drag={type:'move', ...};
  }
});
```
per-zone 리스너 연결 여부와 무관하게 모든 zone 클릭/이동 동작 보장.
