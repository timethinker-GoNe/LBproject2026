# 게임 아이콘 작업 요건

## 파일 형식

- **PNG** (필수)
- JPEG 사용 불가 — 압축 손실 및 투명 배경 지원 안 됨

## 크기

- **256×256 px** 또는 **512×512 px** 권장
- 반드시 2의 거듭제곱 크기로 작업 (128, 256, 512 …)
  - 300×300, 400×400 등 임의 크기는 사용 금지
- 인게임 표시 크기가 작더라도 소스는 256 이상으로 제공

## DPI

- **무관** — 픽셀 수만 맞으면 됨 (72dpi, 300dpi 모두 동일하게 처리됨)

## 배경 / 투명도

- 배경은 **완전 투명** (알파 0) 처리 필수
- 흰 배경 또는 반투명 잔재가 남으면 인게임에서 흰 사각형으로 보임
- Photoshop 저장 시: **파일 → 다른 이름으로 저장 → PNG** 사용
  - "웹용으로 저장(Save for Web)"은 알파채널이 손실될 수 있으므로 사용 금지

## 스타일 가이드

- 기존 게임 내 아이콘 스타일에 맞출 것 (별도 레퍼런스 이미지 첨부 예정)
- AI 생성 이미지 사용 시 배경 제거 후 반투명 픽셀 잔재 여부 반드시 확인

## 납품 체크리스트

- [ ] PNG 형식
- [ ] 256×256 또는 512×512 px
- [ ] 배경 완전 투명 (알파채널 포함)
- [ ] 작업 원본 파일(PSD 등) 별도 보관 및 함께 납품
